pdf_shipping_label_automation

• Python automation for processing multi-page shipping label PDFs, removing unnecessary pages, and streamlining document workflows.

Overview

• Python-based document workflow automation for shipping label processing.

• Developed a Python script that automatically processes multi-page shipping label PDFs by removing unnecessary pages and retaining only required documentation, eliminating manual document cleanup.

• Implemented one-click execution through a Windows batch file for streamlined daily use.

Key Impact

• Reduced multi-page document sets to required pages automatically

• Eliminated repetitive manual PDF editing

• Saved approximately 5–7 minutes per processing run

• Built using Python, pathlib, regex, and pypdf library

Technologies Used

• Python

• pathlib

• regex

• pypdf

• Windows Batch Scripting
